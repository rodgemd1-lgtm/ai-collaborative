# Hexis 6-Lens Evaluation Workflow

**When:** Evaluating a strategic bet.
**Trigger:** `pi-iac hexis`

## The Six Lenses
1. **Consumer** — End consumer in 5 years, unmet needs
2. **Market** — Demand/supply/substitutes reshaping value pool
3. **Technology** — Commodifying vs. differentiating capabilities
4. **Regulatory** — Policy reshaping what's possible
5. **Talent** — Hire/lose/roles that don't exist yet
6. **Capital** — Return profile and financing structures

## V-Score (V0–V10)
| Score | Meaning |
|-------|---------|
| V0–V3 | No AI capability |
| V4–V5 | Experimental/pilot AI |
| V6–V7 | Production AI, no governance |
| V8–V9 | Full platform with governance |
| V10 | AI-native organization |

## Output Template
```
BET: [one sentence]

LENS 1 — CONSUMER
Current: [description]
5yr: [projection]
V-score: V[X]/10

LENS 2 — MARKET
[... same ...]

[... all 6 lenses ...]

VERDICT: [GO | NO-GO | GO-WITH-CONDITIONS]
CONDITIONS: [if applicable]
TARGET: V[X] by [date]
```

## Example Bets
- "Should IAC build the Forefront Platform as SaaS?"
- "Should IAC enter mid-market at $50–500M?"
- "Should IAC partner with RIG for AI platform?"
