# Workflow: Content Creation

## Trigger
Producing blog posts, insights, or social content. Activated by: `Pi IAC content`, "write insight", "draft post"

## Goal
Create persona-aware, framework-first content that passes the Kathleen gate (IA-grade craft) and advances the IAC engagement narrative.

## Steps

### 1. Content Brief (Content Agent)
Define:
- **Format:** Blog post, LinkedIn post, insight article, pitch deck section, email sequence
- **Primary persona:** Kyle (evidence-first), Kathleen (craft-first), Dan (numbers-first), Matthew (technical-first), Jeff (novel-ideas), Rahul (build-first)
- **Framework anchor:** Hexis, IDEALS, or Seven Elements (never generic)
- **Key stat:** One bold stat with source
- **CTA:** What should the reader do next?

### 2. Ideation (Content + Research Agent)
- Pull recent IAC insights / case studies from recon
- Pull competitive intel for differentiation
- Check MiroFish for trending topics in design + AI
- Generate 3 angles; select the one with highest "insight density"

### 3. Drafting Guidelines
- **Mike's voice:** Short, punchy, declarative. "The enterprise AI dollar is claimed. The mid-market AI dollar is not."
- **IAC's voice:** When writing "as IAC," use their language: "Insight to Action," "Live the Problem," "Make it, Now."
- **Formatting:**
  - For blogs: 800–1200 words, subheadings every 150 words, one pull quote
  - For LinkedIn: 150–300 words, 3–5 line breaks, one stat, one question
  - For decks: ≤ 12 slides, bullets only, no paragraphs
- **Stat rule:** Every number needs a source URL or citation

### 4. Kathleen Gate Check
Before any content ships:
- [ ] Clean, modern aesthetic (if visual)
- [ ] Bold stat callouts with sources
- [ ] No paragraph blocks on slides
- [ ] Strong white space
- [ ] Every statistic cited
- [ ] Framework-first (not generic thought leadership)
- [ ] Would this appear in an IA-grade publication?

### 5. Production Pipeline
```
content-draft-[TOPIC]-[DATE].md
├── Brief
├── Research Notes
├── Draft
├── Source List
├── Kathleen Gate Check
└── Publish Plan
```

### 6. Publish & Distribute
- Blog → Web agent for site integration
- LinkedIn → Client agent for scheduling
- Insight → Campaign agent for email sequence
- Deck → Gamma agent for generation

## Never Rules
- Never publish without stat sources
- Never write IAC content as Mike's voice (use IAC language, not personal language)
- Never skip the Kathleen gate on client-facing material
- Never exceed 12 slides for a pitch deck
